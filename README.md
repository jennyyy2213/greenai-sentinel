# GreenAI Sentinel

AI-powered Climate Tipping Point Early Warning System.

## Features
- Global climate risk map
- Real ERA5 data analysis
- Early warning signal detection
- Tipping risk forecasting
- Autonomous global hotspot scanning
- Climate impact exploration

## Data Source
Copernicus ERA5 Reanalysis (ECMWF)

## Technologies
Python, Streamlit, Xarray, Folium, NumPy
